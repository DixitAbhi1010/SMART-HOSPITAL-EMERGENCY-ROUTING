# 🚑 Smart Hospital Emergency Routing (NEXUS)

A smart healthcare system that recommends the best hospital in emergency situations using AI-based decision logic, real-time factors, and routing.

## 🧠 Problem Statement
In emergency situations, patients often waste critical time searching for the right hospital due to:
- Lack of real-time availability data  
- No guidance on best nearby hospital  
- Delayed decision-making  

---

## 💡 Solution
NEXUS provides:
- 🏥 Smart hospital recommendation  
- ⚡ Fast decision-making system  
- 🧠 AI-based routing logic  
- 🗺️ (Optional) Map-based navigation  

---

## ⚙️ Tech Stack
- **Frontend:** React + Vite + Tailwind CSS  
- **Backend:** Node.js + Express  
- **Animations:** Framer Motion  
- **Maps:** Google Maps API (optional)  

---

## 🚀 Features
- AI-based hospital selection  
- Severity-based filtering (Normal / Critical)  
- Distance & availability optimization  
- Modern UI with smooth animations
